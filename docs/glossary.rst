Glossary
========

.. glossary::

    QHAna
        Quantum Humanities Analysis tool.

    Plugin Runner
        A framework that supports the creation of QHAna plugins written in python. (:external+pluginrunner:doc:`Developer documentation <index>`)
    
    QHAna Plugin
    Plugin
        Plugins in QHAna are microservices bundled together with a micro frontend.
        For more information see :external+pluginrunner:doc:`plugins`.
